WEB DEVELOPMENT FLEX-BOX

Project ini merupakan project dari materi WEB DEVELOPMENT FLEX-BOX. Project ini mengharuskan saya untuk membuat poster menggunakan FLEX-BOX dan saya pun memutuskan untuk membuat poster undangan mengikuti pesta yang bertema Halloween. Dapat dilihat bahwa penggunaan FLEX-BOX terdapat pata segi penataan gambar - gambar yang memiliki kotak tersendiri seperti di panel komik.
