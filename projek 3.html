MINI PROJECT WEB BRANDING

Website ini merupakan sebuah mini project yang mengharuskan saya bersama dengan kelompok saya untuk membuat sebuah web branding alias website untuk mempromosikan suatu komunitas. Pada gambar terlihat bahwa kami memutuskan untuk membuat website bernama Canisius Youth Community yakni komunitas untuk mendukung kinerja para kanisian dari bidang seni, workshop, dan teknologi. Website ini juga merupakan mini project yang mencakup semua materi TIK baik itu form, button, javascript, dan yang lain lainnya.
